# WebFramework
基于PyQT5的html套壳框架
基于python的PyQT5中的browser.load方法加载Html

用法介绍
package.json内容如下

{
  "name": "p",
  "version": "1.0.0",
  "size_width": "800",
  "size_height": "400",
  "icon": "test.ico",
  "main_html": "./res/index.html"
}
其中 name对应软件标题
    version对应版本（不填也可以）
    "size_width"和"size_height"分别对应窗口的长，宽
    icon对应窗口的图标
    main_html对应套壳的html文件
